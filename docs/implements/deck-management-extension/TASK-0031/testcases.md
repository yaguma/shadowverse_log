# TASK-0031 テストケース

## 単体テスト（Vitest）

### TC-0031-001: シーズンと対戦日が同じグリッドコンテナ内にある
- **概要**: シーズンフィールドと対戦日フィールドが同じgridコンテナ内にあることを確認
- **期待結果**: シーズンと対戦日の親要素が `grid` クラスを持ち、`md:grid-cols-2` クラスを持つ

### TC-0031-002: 詳細設定の折りたたみが存在する
- **概要**: 詳細設定セクションが `details` 要素として存在することを確認
- **期待結果**: 「詳細設定」というテキストを含む `summary` 要素が存在する

### TC-0031-003: ランクとグループが同じグリッドコンテナ内にある
- **概要**: ランクフィールドとグループフィールドが同じgridコンテナ内にあることを確認
- **期待結果**: ランクとグループの親要素が `grid-cols-2` クラスを持つ

### TC-0031-004: 既存フィールドが正常に表示される
- **概要**: 既存のすべてのフィールドが正常に表示されることを確認
- **期待結果**: 対戦日、シーズン、対戦タイプ、ランク、グループ、使用デッキ、先攻後攻、勝敗、相手デッキが表示される

### TC-0031-005: フォーム送信機能が維持される
- **概要**: レイアウト変更後もフォーム送信機能が正常に動作することを確認
- **期待結果**: 登録ボタンクリックでフォームが送信される
